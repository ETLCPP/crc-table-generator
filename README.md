CRC Table Generator
-------------------------

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://opensource.org/licenses/MIT)

**About**

Generates CRC lookup tables for common CRC algorithms for 8, 16, 32 and 64 bit CRCs.
